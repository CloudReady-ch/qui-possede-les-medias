## Development

[Node.js](https://nodejs.org/) version >= 10 is required.

1. Clone locally the repository.
   ```bash
   git clone https://github.com/DesignandHuman/capitext.git
   cd capitext
   ```
2. Install dependencies.
   ```bash
   npm install
   ```
3. Start development server.
   ```bash
   npm run start
   ```
